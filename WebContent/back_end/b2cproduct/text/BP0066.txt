優點
郵件伺服器防護
Kaspersky Security for Mail Server 可在最新版的各大郵件與協同作業平台保護郵件，包括 Microsoft Exchange、IBM Lotus Domino 以及 Linux 的郵件伺服器。

多層式垃圾郵件過濾
智慧型的雲端垃圾郵件過濾功能可保護 Microsoft Exchange 與 Linux 郵件伺服器，並且能夠透過即時防護功能顯著減少垃圾訊息佔用的流量。

系統資源使用方式最佳化
Kaspersky Security for Mail Server 擁有全新的防毒引擎、精心設計的負載平衡以及最佳化的防毒掃描功能，並且能夠排除特定物件不加以掃描。本產品不但可維持效能，更可最小化防毒掃描所消耗的資源。

確保穩定的安全防護效能
系統關機時即會自動重新啟動，協助您維持穩定的安全防護效能，讓您安心查找故障原因。

增強資料保護與促進法規遵循
Kaspersky Security for Mail Server 會分析外寄電子郵件的內容，並自動封鎖包含企業機密資料或員工機密資訊的訊息與附件。 除了掃描包含在預先安裝詞彙表或系統管理員自訂詞彙表中的字詞，系統也會掃描結構化資料。*

*資料遺失防護功能僅適用於 Microsoft Exchange Servers，且授權需另購
